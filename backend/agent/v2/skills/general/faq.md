## 技能概述

解答用户关于系统功能、使用方法、常见问题的咨询。

## 输入规范

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| question | string | 是 | 用户的问题 |

## 前置条件

- [ ] 无需特殊前置条件

## 执行逻辑

### 步骤1: 问题分类

将用户问题归类：
- **功能咨询**: 系统能做什么
- **使用指导**: 某功能如何使用
- **问题排查**: 遇到问题怎么办
- **账号相关**: 账号、隐私、安全

### 步骤2: 知识库匹配

从FAQ知识库中匹配最佳答案：
```
知识库分类:
- 系统功能: 餐厅搜索、菜品推荐、电影推荐...
- 使用方法: 如何搜索、如何收藏、如何评价...
- 常见问题: 定位不准、加载慢、无结果...
- 账号隐私: 数据安全、隐私政策...
```

### 步骤3: 答案组织

根据匹配结果组织回复：
- 直接回答问题
- 提供操作步骤（如有）
- 给出相关功能引导

### 步骤4: 补充建议

如果问题较复杂，建议用户：
- 查看详细帮助文档
- 联系客服支持

## 约束条件

1. **准确性**: 答案必须基于实际功能
2. **简洁性**: 回答简明扼要
3. **引导性**: 适时引导用户体验功能
4. **边界**: 不回答超出系统范围的问题

## FAQ 知识库

### 系统功能

**Q: 你们能做什么？**
A: 我是您的生活助手，可以帮您：
- 搜索附近餐厅、推荐美食
- 推荐电影、查询影院
- 规划休闲活动
- 回答生活相关问题

**Q: 怎么找餐厅？**
A: 直接告诉我您的需求即可，比如：
- "附近有什么好吃的？"
- "找一家川菜馆"
- "人均100的餐厅推荐"

### 使用方法

**Q: 如何获取更精准的推荐？**
A: 您可以提供更多信息：
- 位置：已开启定位或告诉我地址
- 偏好：菜系、价格、环境要求
- 场景：聚餐、约会、商务

**Q: 推荐的餐厅不喜欢怎么办？**
A: 您可以：
- 说"换一批"获取新的推荐
- 提供更具体的筛选条件
- 告诉我您的具体偏好

### 常见问题

**Q: 定位不准确怎么办？**
A: 请检查：
1. 手机定位权限是否开启
2. 是否在室内影响信号
3. 可手动输入地址

**Q: 搜索没有结果？**
A: 可能原因：
1. 搜索范围太小，尝试扩大范围
2. 筛选条件太严格，适当放宽
3. 该区域确实缺少相关服务
